- Para compilar usar el comando make en el directorio raíz.

- Para ejecutar usar el comando ./bin/ProgramaImagen [archivo pgm deseado]

	Ejemplo de ejecución: ./bin/ProgramaImagen ./imagenespgm/cameraman.pgm

- Para eliminar los archivos generados al compilar usar el comando make clean en el directorio raíz.

- En la carpeta doc se encuentra la memoria de la práctica en formato pdf.

- En la carpeta "Ejemplos" se incluyen los ejemplos usados en la documentación, tanto los .pgm
  originales como los modificados por las funciones del programa.
